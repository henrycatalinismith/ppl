### Name

ppl ls - List all contacts

### Synopsis

    ppl ls

### Description

List all the contacts in the address book.

### Examples

    $ ppl ls
    jdoe: John Doe <jdoe@example.org>
    kdoe: Kent Doe <kdoe@example.org>
