ppl Quick Start

Installation

sudo gem install ppl

 

ppl is distributed via rubygems. Typically this will already be installed on your system.

Create An Address Book

ppl init ~/contacts

 

Just like creating a Git repo.

Add Some Contacts

cd ~/contacts
ppl add alice "Alice Adams"
ppl add bob "Bob Brown"
ppl add eve "Eve Edwards"

 

Store Some Contact Details

ppl email alice alice@example.org
ppl phone bob 0123456789
ppl org eve "Red Hat"
